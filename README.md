# Initial PC Setup (Мастер первоначальной настройки)

**Версия 0.1 (временная, находится в активной разработке)**

## О проекте

PowerShell-скрипт с графическим интерфейсом WPF, предназначенный для быстрой первоначальной настройки нового компьютера в корпоративной среде.  
Скрипт автоматизирует рутинные операции:

- Резервное копирование текущих сетевых настроек
- Установка статического IP-адреса, маски, шлюза и DNS-серверов
- Переименование компьютера
- Ввод в домен Active Directory
- Ведение лога всех действий

Всё это выполняется через удобное графическое окно, которое можно настраивать с помощью Visual Studio (WPF).

## Текущие возможности

- ✅ Создание бэкапа текущих параметров IP на рабочем столе
- ✅ Настройка статического IPv4 и DNS
- ✅ Смена имени компьютера
- ✅ Присоединение к домену Active Directory
- ✅ Логирование всех операций
- ✅ Графический интерфейс (WPF) с возможностью редактирования в Visual Studio

## Системные требования

- Windows 10/11 (или Windows Server с GUI)
- PowerShell 5.1 (встроен в Windows)
- Запуск от имени администратора
- Учётные данные с правами на добавление ПК в домен
- Доступность контроллера домена по сети (DNS должен быть настроен)

## Инструкция по использованию

1. Скачайте файл `InitialSetup.ps1`.
2. Запустите PowerShell от имени администратора.
3. Временно разрешите выполнение скриптов:  
   `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process`
4. Перейдите в папку со скриптом и выполните:  
   `.\InitialSetup.ps1`
5. Заполните все поля в появившемся окне и нажмите **OK**.
6. При появлении окна введите учётные данные администратора домена.
7. Дождитесь завершения операций и перезагрузите компьютер.
